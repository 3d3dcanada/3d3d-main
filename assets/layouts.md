# 3D3D Creative Layout System

## Visual Direction
- Primary background: deep graphite `#111214`
- Primary accent: marine teal `#40C4C4`
- Secondary accent: signal coral `#E8665A`
- Light surface: soft bone `#F6F7FA`
- Utility text: warm gray `#666666`
- Typography direction: bold condensed display for headlines, mono/tech face for labels, readable sans for body
- Photo style: real workshop, real deck hardware, real parts, real hands; avoid generic CGI and stock “futuristic printing” visuals
